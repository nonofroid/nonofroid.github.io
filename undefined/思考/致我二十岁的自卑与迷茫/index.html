致我二十岁的自卑与迷茫

发表于 2022-05-14 更新于 2022-05-20 分类于 思考 热度： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

今天看了一个在澳洲墨尔本留学的UP主的视频。

本意并非传播焦虑，当下的行动、乃至人生的意义，现阶段的我花了很多时间思考。我想要每时每刻思考着我现在应该做什么，才不会陷入虚无的境地。

二十岁的我，开始想着宇宙天地与时间是什么。不过暂且先不论宇宙、时间这类宏大的科学问题(托尔斯泰笑了)。如果把目光放到个体身上，我为什么要学习我现在的专业?我应该怎么做才能把它做得更好?除了学习专业之外，我应该做些什么以更好地过渡到现实社会中？我以后的目标是什么？我对于成功的定义是什么？学习的过程中我受到过什么影响？

…

在成长过程中，“自卑”始终环绕着我。

这种强烈的自卑感从何而来？

觉得自己专业知识没有达到预期，觉得自己课外知识面不够宽阔，觉得自己外貌与身材不是很好，对自己的赚钱能力感到担忧。这种自卑感几乎都是与他人对比而产生：为什么有的人成绩那么优秀？为什么有的人“样样精通”？为什么有些人天生面容姣好(而受到他人的欢迎)？为什么有的人可以通过投资获得那么多物质财富？…

而集体主义的重压更让我这种普通人喘不过气来。自卑所带来的不安全感，也从来没有让我考虑过开展所谓的感情关系，这种持久潜意识似乎让我失去了与他人建立亲密关系的能力。其次，我也并不在乎所谓社会标准与世俗眼光。

宇宙已经发展了整整快140亿年，纵观人类渺小的就像一粒不起眼的尘埃。人生在世短短百余年，所以为什么不好好让自己探索内心、探索外在，而要让自己拘泥于原生家庭、后代传承、快餐感情等无聊而又琐碎的社会标准呢？现代社会的基本特征就是理性，这种理性打破了过往人类群体所共同信仰的统一秩序。因此，个人价值观的冲突、社会问题的对立在当今时代是一个非常严重的问题。我认识到每个人的出生起点、原生家庭、社会环境都是不同的，所以我没有必要一味地拿我自己对标他人、他人有的我也要有。这种想法我过去曾经深深烙印在脑海里，现在看来十分幼稚与愚钝。我想我只需要偶尔停下来好好想想自己究竟想要什么，自己现阶段不需要什么，然后在继续做事就行了。

举个例子，现阶段社会要求一个人的标准大致是年少时学习-30岁左右工作/成家立业-养老。一套既定的社会标准，为了人的延续发展和社会的稳定，是必要的。我并非不认同这种生活方式，而是这种既定的标准仅仅只是一个通用的模板。还是我之前的想法，在多元化的社会背景下，每个人都是不同的。一个人即使30多岁，可能还是想继续发展事业、拓宽自身的眼界而不着急成家立业；一个人可能是独身主义/不婚主义者；一个人可能是同性恋、双性恋、跨性别群体…我不认同一个人要违背自己的意志生硬地执行社会给他的标准这种做法。让自己感到现在做的事是有意义的最为重要。什么是有意义的？每个人内心的标准不同，可能是读更多的书、可能是去更多的地方、可能是拥有更大的名声、可能是赚更多的钱…

种种内在经验告诉我，不能总是把思维置于虚无的未来中，那只是思维的幻想。一味地假设未来会发生什么，而不顾及当下，是一种效益比十分低下的行为。

如何避免思维上的“三天打鱼与两天晒网”，是一种需要长期锻炼的思维方式。我们总是不可避免地多多少少会受到环境的影响。告诉自己想要的是什么，结交能够互相给予帮助、具有上进与事业心的朋友，为自己的思维提供一种积极、友好的环境。做好了当下，才能立足于今后你想要做的事。

在生活中不要让你的思维服务器宕机。缺乏思维的理性安排，一切都会陷入无序，进而陷入一种危险的循环。尽可能地保持自己头脑正常运转，简单厘清现在自己应该做什么，防止自己陷入无尽的拖延。

五月天那首《盛夏光年》里“难道成长是人毕竟的溃烂”这句歌词真的是人必经的道路吗？人活一生，痛苦相随，我没有太多朋友可以倾诉，也不想告诉家人我真实的想法(他们大概是不会轻易理解我的吧，反而还会太担心我的精神状态)，所以我更想听一会本我的想法，清楚自己现阶段想要的和暂时不需要的，尽情纵情山水与天地之间。

走，就这么自由地走吧，绝不留恋、义无反顾地走吧。

学习，阅读，投资，运动，吉他，摄影。
我本应该是热爱他们的。
我是一名悲观主义者，我也是一名完美主义者。我一直在力图避免所有尴尬的时刻，我无法容忍与他们共存。我本应该可以达到完美的，但是我却亲手近乎重塑了它。
我想做得更好，无论是现实生活，还是在虚拟世界中。
